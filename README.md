# Hotel Booking Website

 Overview
The Hotel Booking Website is a web-based application developed to simplify and 
automate the process of searching, booking, and managing hotel accommodations. 
It connects three primary users — Admin, Hotel, and User — within a single 
integrated platform, eliminating the inefficiencies of traditional manual booking 
methods and providing a seamless digital reservation experience.

Objective
The main objective of this project is to automate the hotel booking process by 
providing a centralized web platform. It enables users to browse hotels, check 
room availability, and complete bookings online, while hotels manage reservations 
and payments efficiently, and admins oversee the entire system.

Features
- User registration and login
- Browse and search available hotels
- View room details, amenities, and pricing
- Real-time booking request and status updates
- Hotels can accept or reject booking requests
- Payment enabled only after booking acceptance
- Admin, Hotel, and User role-based modules
- Secure authentication with encrypted passwords
- Booking history and payment tracking

Technologies Used
- Frontend: HTML, CSS, JavaScript, Bootstrap, Tailwind CSS
- Backend: Python (Flask)
- Database: MySQL
- Tools: PyCharm Community Edition

System Description
The system maintains a centralized MySQL database storing all user, hotel, room, 
booking, and payment records. When a user submits a booking request, it is sent 
to the respective hotel for review. The hotel can accept or reject the request in 
real time. Upon acceptance, the payment module is activated, allowing the user to 
complete the transaction securely. The admin monitors all activities through a 
dedicated dashboard, ensuring smooth and transparent system operation.

Modules
- Admin Module: Manages hotels, monitors users, and oversees all bookings and payments
- Hotel Module: Adds room details, manages booking approvals, and views payments
- User Module: Registers, browses hotels, makes bookings, and completes payments

Installation
1. Install Python and required libraries
2. Set up MySQL database named Hotel_Booking_Website
3. Configure database connection in the project
4. Run the Flask application
5. Open browser at http://localhost:5000

 Advantages
- Eliminates manual booking process
- Real-time booking status updates
- Secure and transparent payment handling
- Easy to use for all types of users
- Centralized control for admins and hotels

Future Enhancements
- Mobile application support

Conclusion
The Hotel Booking Website is an effective digital solution for modernizing the 
hotel reservation process. It uses automation, role-based access, and real-time 
updates to ensure efficient booking management and an improved experience for 
both travelers and hotel administrators.
